# Banco de Capacitores - Calculadora

Este programa en Python calcula los parámetros de un banco de capacitores, incluyendo el número de capacitores requeridos, la capacidad total real del banco, las potencias (activa, reactiva y aparente), la carga, la tensión, el amperaje, la velocidad de descarga y la velocidad de carga del capacitor.

## Requisitos

- Python 3.x

## Uso

1. Clona o descarga este repositorio en tu máquina local.

2. Abre una terminal y navega al directorio donde se encuentra el archivo `banco_capacitores.py`.

3. Ejecuta el siguiente comando para iniciar el programa:

   ```bash
   python banco_capacitores.py
4.Sigue las instrucciones en la terminal e ingresa los valores requeridos (capacidad total, capacidad individual, voltaje, resistencia de descarga y resistencia de carga).

5. El programa calculará y mostrará los resultados por pantalla.


Ejemplo
Aquí tienes un ejemplo de los valores que puedes utilizar:
Ingrese la capacidad total deseada en microfaradios: 10000
Ingrese la capacidad individual de cada capacitor en microfaradios: 100
Ingrese el voltaje en voltios: 220
Ingrese el valor de resistencia de descarga en ohmios: 1000
Ingrese el valor de resistencia de carga en ohmios: 500
